 EcoPainel - Monitoramento Verde

Este projeto foi desenvolvido para o Concurso Agrinho 2026, dentro da categoria Programação, abordando o tema "Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente".

Sobre o Projeto
O EcoPainel simula um sistema administrativo real (*Dashboard*) de uma propriedade agrícola localizada em Cascavel-PR. O sistema reúne cartões de métricas agroambientais e um simulador de expansão florestal: o produtor projeta o plantio de mudas de árvores nativas e o painel atualiza instantaneamente o sequestro de carbono e gera novos créditos ambientais, somando-os ao saldo global da fazenda. Há também uma tabela focada na rastreabilidade e certificação de safra.

 Tecnologias e Declaração de Uso de IA
A aplicação utiliza uma arquitetura avançada de layout baseada em **CSS Grid** e **Flexbox** para o formato de painel administrativo, estruturação em **HTML5** e **JavaScript** orientado a eventos para a reatividade dos dados.

Como parte do processo de letramento digital orientado, declara-se que o uso de ferramentas de Inteligência Artificial ocorreu de forma assistida para fins de:
1. Otimização do design da interface (regras de posicionamento da barra lateral e tabelas).
2. Refinamento de funções Javascript para formatação de moedas e casas decimais (`toFixed`).
A estrutura do painel, a contextualização regional e a correlação das métricas agrícolas foram criadas e defendidas pelo estudante.
